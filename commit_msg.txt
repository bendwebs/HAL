feat: add standalone image generation page

Creates a new /generate page for Stable Diffusion image generation,
separate from the chat system. Features include:

- Direct prompt input without LLM middleman
- Advanced settings (dimensions, steps, CFG scale, sampler, seed)
- Dimension presets for common sizes
- Batch generation (1-4 images)
- SD status display with start/stop controls
- Image gallery showing all generated images
- Lightbox view for full-size images
- Download and delete functionality
- Responsive layout for desktop and mobile

Backend changes:
- Added POST /api/images/generate endpoint
- Added Pydantic models for request/response validation

Frontend changes:
- Added imageGen API functions
- Added Generate link to sidebar and mobile nav
- Created full-featured generation page
